# SQL Projects

This folder contains data analysis projects using SQL to extract, transform, and analyze data from relational databases.

## Skills Demonstrated
- Data querying using SELECT, WHERE, and ORDER BY
- Data aggregation with GROUP BY and HAVING
- Table relationships using JOIN
- Subqueries and basic performance analysis

## Projects
- Sales Data Analysis
- Customer Behavior Analysis
- Business Performance Reporting

These projects focus on answering business questions and generating insights to support data-driven decision-making.
